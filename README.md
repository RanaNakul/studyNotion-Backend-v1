## Introduction

StudyNotion is an application designed to help users manage their study schedules, track progress, and collaborate with peers. This repository contains the backend code written in JavaScript.

## Features

- User authentication and authorization
- CRUD operations for study notes
- Scheduling and reminders
- Collaboration tools

## Installation

To install and set up the project locally, follow these steps:

1. Clone the repository:
   ```sh
   git clone https://github.com/RanaNakul/studyNotion-Backend-v1.git
   cd studyNotion-Backend-v1
   ```

2. Install dependencies:
   ```sh
   npm install
   ```

3. Set up environment variables:
   Create a `.env` file in the root directory and add the necessary environment variables. Refer to `.env.example` for the required variables.

4. Start the server:
   ```sh
   npm start
   ```
